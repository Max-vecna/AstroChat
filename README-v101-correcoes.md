# AstroChat v101 - Fundo do chat acompanha o tema

## Arquivos alterados

- `app.js`
- `style.css`
- `service-worker.js`

## O que foi corrigido

1. Ao trocar o tema no perfil, agora também muda o fundo da área de conversa.
2. O fundo da lista de mensagens passa a usar a paleta do tema escolhido.
3. O fundo do rodapé onde fica o campo de digitação também acompanha o tema.
4. O campo de texto e os botões auxiliares do compositor receberam fundo baseado nas cores do tema.
5. No tema Galáctico, o fundo da conversa foi ajustado sem os três pontos/padrão pontilhado antigo.
6. Cache do PWA atualizado para `astrochat-cache-v101`.
7. Versão exibida no perfil atualizada para `AstroChat v101`.

## Como instalar

1. Substitua os arquivos da raiz do site por estes arquivos atualizados.
2. Publique o hosting novamente.
3. No celular, feche e abra o PWA. Se ainda carregar a versão antiga, limpe os dados do site ou reinstale o app.
